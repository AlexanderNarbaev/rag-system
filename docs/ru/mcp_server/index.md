# MCP-сервер

Сервер Model Context Protocol (MCP) предоставляет инструменты RAG-системы для MCP-совместимых клиентов, таких как
OpenCode, Claude Desktop и другие AI-ассистенты.

## Обзор

MCP-сервер предоставляет:

- **Инструменты** — поиск по базе знаний, запрос версий документов, проверка работоспособности системы
- **Ресурсы** — доступ к проиндексированным документам, ADR и конфигурации
- **Промпты** — готовые шаблоны промптов для типовых RAG-взаимодействий

## Архитектура

```
┌──────────┐   MCP Protocol (STDIO/HTTP)   ┌──────────────┐
│  Клиент   │ ◄──────────────────────────► │  MCP Server  │
│ (OpenCode,│                               │  (FastMCP)   │
│  Claude)  │                               └──────┬───────┘
└──────────┘                                       │
                                                    │ OpenAI API
                                                    ▼
                                            ┌──────────────┐
                                            │  RAG Proxy   │
                                            │    :8080     │
                                            └──────────────┘
```

## Конфигурация

### Интеграция с OpenCode

Добавьте в `opencode.json`:

```json
{
  "mcpServers": {
    "rag-system": {
      "command": "python",
      "args": ["-m", "mcp_server.server"],
      "cwd": "/path/to/rag-system",
      "env": {
        "RAG_PROXY_URL": "http://localhost:8080/v1"
      }
    }
  }
}
```

### Интеграция с Claude Desktop

Добавьте в `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "rag-system": {
      "command": "python",
      "args": ["-m", "mcp_server.server"],
      "cwd": "/path/to/rag-system"
    }
  }
}
```

## Доступные инструменты

| Инструмент              | Описание                                                     |
|-------------------------|--------------------------------------------------------------|
| `search_knowledge_base` | Гибридный поиск по всем проиндексированным документам        |
| `get_document`          | Получение конкретного документа по ID                        |
| `list_sources`          | Список доступных источников данных и статус их синхронизации |
| `health_check`          | Проверка работоспособности прокси и зависимостей             |

## Режимы транспорта

- **STDIO** — режим по умолчанию для десктоп-клиентов (Claude Desktop, OpenCode)
- **Streamable HTTP** — для удалённых или серверных клиентов

См. [Интеграция с OpenCode](../guides/integration-opencode.md) для подробных инструкций по настройке и примеров
использования.
