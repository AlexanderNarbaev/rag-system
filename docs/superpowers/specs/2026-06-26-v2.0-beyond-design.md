# v2.0 — Beyond v1.0: Self-Correcting RAG, Agentic Tools, Multi-Language

> Date: 2026-06-26 | Status: Approved

## Scope
Three parallel workstreams for the next generation of capabilities:

### A. Self-Correcting RAG Level 5 (6 features)
1. **HyDE** — hypothetical document generation → embedding → retrieval
2. **CRAG Evaluator** — enhanced retrieval scoring with per-document grading
3. **Self-Reflection Loops** — multi-hop reasoning with verification steps
4. **Answer Verification** — claim-level entailment checking
5. **Hallucination Detection** — identify unsupported claims
6. **Hallucination Rate Benchmark** — target < 5%

### B. Agentic Tools (5 features)
1. **Function Calling API** — expose retrieval as a tool for LLM
2. **Live Confluence queries** — direct API integration
3. **Live Jira queries** — direct API integration
4. **Live GitLab queries** — direct API integration
5. **Tool Registry** — pluggable tool system

### C. Multi-Language (4 features)
1. **DE scaffolding** — German prompts, tokenization, retrieval
2. **FR scaffolding** — French prompts, tokenization, retrieval
3. **ZH scaffolding** — Chinese prompts, tokenization, retrieval
4. **Cross-lingual retrieval benchmarks**

### D. Documentation
Update all docs for v2.0, maturity Level 5, updated scores.

## Principles
- Air-gapped first
- Graceful degradation
- TDD: test → code → commit → push
- Agentic Tools: tool calls are optional, graceful fallback
- Multi-language: language auto-detection, fallback to EN
